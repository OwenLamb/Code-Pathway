Lesson 2: Introduction to Conditional Statements

In programming, conditional statements let you execute specific code based on certain conditions, adding dynamic behavior to your scripts. They allow your code to make decisions and respond differently depending on the situation. The basic structure in Python is an if statement, which checks whether a condition is True or False.

Here is an example: 

state = True

if state == True:
    print("Hello World")
else:
    print("No dice")

In this code:

  • state is a boolean variable, which can be either True or False.
  • The if statement checks if state is True. If it is, the code inside the if block executes, printing "Hello World".
  • If state is False, the code inside the else block runs, printing "No dice".

You can use different conditions other than == (equals). There are <= less than or equal to, > greater than, etc. You can also add additional conditions with elif (short for "else if"), which checks another condition if the first one isn’t met. Here’s an example: 

state = False
alternate_state = True

if state == True:
    print("Hello World")
elif alternate_state == True:
    print("Hello Universe")
else:
    print("No dice")

In this example:

  • If state is True, it prints "Hello World".
  • If state is False but alternate_state is True, it prints "Hello Universe".
  • If both state and alternate_state are False, it prints "No dice".

Conditional statements are powerful tools for creating flexible and responsive code.
